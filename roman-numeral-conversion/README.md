Roman Numeral Conversion

```python
number = int(input('Enter a number, must be less than 10'))

if (number == 1):
    print('I')
elif (number == 2):
    print('II')
elif (number == 3):
    print('III')
elif (number == 4):
    print('IV')
elif (number == 5):
    print('V')
elif (number == 6):
    print('VI')
elif (number == 7):
    print('VII')
elif (number == 8):
    print('VII')
elif (number == 9):
    print('IX')
elif (number == 10):
    print('X')
else:
    print('ERROR: the interger is out of range')
```
